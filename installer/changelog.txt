🚀 APO Studio v1.0.6

Mise à jour avec mécanisme d'arrêt réel et améliorations de l'interface utilisateur.

✨ Nouveautés
- Bouton Stop avec annulation réelle du pipeline de traitement
- Bouton Relancer pour relancer les vidéos en attente
- Boutons Nouveau Projet et Traitement par lot côte à côte
- Centrage du dashboard et meilleure utilisation de l'espace

🤖 Annulation intelligente
- Un mécanisme threading.Event propage l'annulation à travers tout le pipeline
- Vérification de l'arrêt à chaque étape : transcription, YouTube, thumbnail, sauvegarde
- Arrêt propre sans corrompre les fichiers en cours de création
- Exception Cancelled gérée séparément des erreurs classiques

🎨 Interface utilisateur
- Nouveau layout avec boutons d'action alignés horizontalement
- Bouton Stop désactivé par défaut, activé uniquement lors d'un traitement
- Couleurs unifiées pour tous les boutons d'action
- Suppression de l'espace vertical inutile entre les widgets

🔧 Corrections
- Fix du bouton Stop qui n'annulait pas réellement le traitement
- Fix du centrage du dashboard qui n'utilisait pas tout l'espace disponible
- Fix de la détection de mise à jour : gestion des tags GitHub en majuscule (V1.0.5)
- Fix du bouton Relancer apparaissant sans erreur

⚙️ CI/CD
- Ajout du workflow GitHub Actions pour la compilation automatique
- Build de l'installateur via Inno Setup à chaque tag de version
- Création automatique de la release avec l'installeur
