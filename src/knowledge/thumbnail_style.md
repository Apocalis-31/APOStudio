# Miniatures

Les miniatures doivent représenter le moment le plus marquant de la vidéo.

Ne jamais inventer une scène inexistante.

Éviter les faux clickbaits.

Le texte doit être très court.

Maximum quatre mots.

Les couleurs doivent être fortes.

L'ambiance doit attirer immédiatement le regard.

Le personnage principal doit être clairement identifiable lorsque c'est possible.
