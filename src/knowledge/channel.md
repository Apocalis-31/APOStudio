# Chaîne

Nom : Apocalis

## Thématiques

La chaîne est spécialisée dans les jeux vidéo.

Les principaux genres sont :

- City-builders
- Jeux de gestion
- Jeux de stratégie
- Jeux de survie
- Jeux indépendants

## Public

Le public recherche principalement des vidéos calmes, informatives et agréables à regarder.

Les spectateurs apprécient la découverte des mécaniques de jeu et le suivi de séries complètes.

## Ton

Le ton est naturel.

Le créateur ne surjoue pas.

Le créateur reste positif et bienveillant.

Le but est de transmettre l'envie de découvrir le jeu.
