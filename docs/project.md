# 📊 Suivre le traitement

Pendant le traitement d'une vidéo, APO Studio affiche en permanence l'état d'avancement du projet grâce au tableau de bord.

Le workflow est composé de cinq étapes :

### 📁 Création du projet

Création de la structure du projet et préparation des fichiers nécessaires au traitement.

---

### 🎤 Transcription Whisper

La vidéo est transcrite automatiquement avec Whisper.

Cette étape est généralement la plus longue du traitement.

---

### 🤖 Génération YouTube

L'intelligence artificielle analyse la transcription afin de générer les éléments nécessaires à la publication :

- le titre ;
- le sous-titre ;
- l'introduction ;
- le prompt de miniature.

---

### 🖼 Génération miniature

APO Studio sélectionne automatiquement les meilleures captures d'écran de la vidéo afin de préparer les miniatures.

---

### 💾 Sauvegarde

Les fichiers générés sont enregistrés dans le dossier du projet.

Le projet est alors prêt à être utilisé.

# 📈 Informations de session

Le tableau de bord affiche également plusieurs informations utiles pendant le traitement :

- ⏱ Temps écoulé depuis le début de la session.
- ⏳ Temps restant estimé.
- 🎬 Nombre de vidéos traitées.
- 📚 Nombre de vidéos restantes.
- 🏁 Heure de fin estimée.

Ces informations sont automatiquement mises à jour tout au long du traitement.
