# 🖼️ Génération des miniatures

APO Studio prépare automatiquement les éléments nécessaires à la création de vos miniatures YouTube.

L'objectif est de vous faire gagner du temps tout en conservant votre propre identité graphique.

---

# 🎬 Comment fonctionne la génération ?

Une fois la transcription et la génération IA terminées, APO Studio analyse la vidéo afin de sélectionner les captures d'écran les plus pertinentes.

En parallèle, l'intelligence artificielle génère un prompt décrivant la miniature à réaliser.

Vous disposez ainsi :

- 🖼 des meilleures captures de la vidéo ;
- 🤖 d'un prompt de génération adapté au contenu ;
- 📝 d'un titre cohérent avec la vidéo.

---

# 🎨 Une aide à la création

APO Studio ne crée pas directement votre miniature.

Le logiciel prépare tous les éléments nécessaires afin de vous permettre de créer rapidement un visuel dans votre outil de génération d'images préféré.

Vous gardez ainsi le contrôle total sur votre direction artistique.

---

# 💡 Bonnes pratiques

Pour obtenir les meilleurs résultats :

- choisissez une capture montrant un moment fort de la vidéo ;
- privilégiez un personnage ou un élément important au centre de l'image ;
- utilisez un texte court et facilement lisible ;
- conservez une identité graphique cohérente entre vos épisodes.

---

# 📂 Où retrouver les fichiers ?

Tous les éléments générés sont enregistrés dans le dossier du projet.

Vous y retrouverez notamment :

- les captures sélectionnées ;
- le prompt de génération ;
- les informations générées par l'IA.

Vous pouvez utiliser ces fichiers directement dans votre logiciel de création d'images.

---

# 📌 À savoir

La génération de miniatures est entièrement optionnelle.

Vous pouvez utiliser uniquement les captures, uniquement le prompt, ou créer votre propre miniature selon vos préférences.
