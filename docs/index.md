# 🚀 Guide de démarrage

Bienvenue dans **APO Studio** !

APO Studio automatise la préparation de vidéos YouTube à partir de vos VOD Twitch.

Le logiciel prend en charge l'ensemble du workflow :

- 🎬 Création du projet
- 🎤 Transcription Whisper
- 🤖 Génération du contenu YouTube avec une IA
- 🖼 Génération des miniatures
- 💾 Sauvegarde du projet

---

# Premier lancement

Avant de commencer, vérifiez que :

- FFmpeg est correctement installé.
- Whisper est disponible (le modèle sera téléchargé automatiquement lors de la première utilisation).
- Un fournisseur IA est configuré (GLM, OpenAI, Claude ou Ollama).

---

# Créer un projet

Cliquez sur **Nouveau projet**.

Sélectionnez votre vidéo.

APO Studio créera automatiquement un dossier contenant tous les fichiers nécessaires au montage et à la publication.

---

# Traitement par lot

Vous pouvez également sélectionner plusieurs vidéos.

Chaque vidéo sera traitée automatiquement l'une après l'autre.

Le tableau de bord affiche :

- la vidéo en cours,
- la progression,
- le temps écoulé,
- le temps restant estimé,
- l'heure de fin prévue.

---

# Accéder à vos projets

Une fois un projet créé, vous pouvez ouvrir son dossier à tout moment.

Il suffit de cliquer sur l'icône **📂** située en haut à gauche de la fenêtre principale.

APO Studio ouvrira automatiquement le dossier contenant tous vos projets.

Chaque projet possède son propre dossier avec :

- 🎬 La vidéo originale
- 🎤 Les fichiers de transcription
- 🤖 Les contenus générés par l'IA
- 🖼 Les miniatures
- 💾 Les fichiers de travail

Vous pouvez ainsi accéder facilement à tous les éléments nécessaires au montage et à la publication de votre vidéo.
