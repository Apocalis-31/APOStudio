# ⚙️ Paramètres

La fenêtre **Paramètres** permet de personnaliser le fonctionnement d'APO Studio.

Vous pouvez modifier ces options à tout moment. Les changements seront automatiquement pris en compte pour les prochains traitements.

---

# 📂 Dossier des projets

Chaque vidéo traitée possède son propre dossier contenant :

- 🎬 les fichiers du projet ;
- 🎤 la transcription ;
- 🤖 les contenus générés par l'IA ;
- 🖼 les ressources pour les miniatures.

Vous pouvez ouvrir ce dossier rapidement grâce au bouton **📂** situé dans la barre supérieure de la fenêtre principale.

---

# 🤖 Fournisseur IA

APO Studio prend en charge plusieurs fournisseurs d'intelligence artificielle.

Selon le fournisseur sélectionné, certaines informations supplémentaires peuvent être demandées, comme une clé API ou un modèle.

Le changement de fournisseur est immédiat et n'affecte pas les projets déjà créés.

---

# 🎤 Whisper

Les paramètres de Whisper permettent d'adapter la transcription à votre configuration.

Lors de la première utilisation, les modèles nécessaires peuvent être téléchargés automatiquement.

Cette opération n'est effectuée qu'une seule fois.

---

# 🎬 FFmpeg

APO Studio utilise FFmpeg pour certaines opérations vidéo.

Assurez-vous que FFmpeg est correctement installé et accessible si vous utilisez des fonctionnalités nécessitant un traitement vidéo.

---

# 💾 Sauvegarde

Toutes les modifications des paramètres sont enregistrées automatiquement.

Vous n'avez aucune action supplémentaire à effectuer avant de fermer la fenêtre.

---

# 💡 Conseil

Avant votre premier traitement, prenez quelques minutes pour vérifier :

- 📂 le dossier des projets ;
- 🤖 votre fournisseur IA ;
- 🎤 la configuration de Whisper.

Une fois ces paramètres configurés, APO Studio est prêt à traiter vos vidéos.
