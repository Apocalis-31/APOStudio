# 🤖 Génération IA

APO Studio utilise une intelligence artificielle pour générer automatiquement les éléments nécessaires à la publication de vos vidéos YouTube.

Selon le fournisseur choisi, l'IA peut générer :

- 📝 L'introduction de la vidéo
- 🎬 Le titre YouTube
- 📄 Le sous-titre
- 🖼 Le prompt de génération des miniatures

---

# ⚙️ Choisir un fournisseur IA

APO Studio est compatible avec plusieurs fournisseurs :

### 🌐 OpenAI

Utilise les modèles GPT.

Recommandé pour une qualité de génération élevée et des réponses rapides.

---

### 🌟 GLM

Alternative performante offrant un excellent rapport qualité/prix.

Parfait pour une utilisation quotidienne.

---

### 🧠 Claude

Très performant pour les textes longs et les résumés.

Peut produire des descriptions particulièrement naturelles.

---

### 💻 Ollama

Fonctionne entièrement en local.

Aucune donnée n'est envoyée sur Internet.

Nécessite un modèle installé sur votre ordinateur.

---

# 🔄 Changer de modèle

Le fournisseur ainsi que le modèle utilisé peuvent être modifiés à tout moment depuis les paramètres d'APO Studio.

Les modifications seront prises en compte pour les prochains projets.

---

# 💡 Quel fournisseur choisir ?

Si vous découvrez APO Studio :

- 🌟 GLM est un excellent choix pour commencer.
- 🌐 OpenAI offre une excellente qualité de génération.
- 🧠 Claude est particulièrement adapté aux textes longs.
- 💻 Ollama est recommandé si vous souhaitez travailler entièrement en local.

---

# ⚠️ En cas d'erreur

Si la génération échoue :

- Vérifiez votre connexion Internet (sauf avec Ollama).
- Vérifiez que votre clé API est correctement configurée.
- Pour Ollama, assurez-vous que le serveur est démarré et que le modèle est installé.

Les erreurs détaillées sont affichées dans la console du logiciel.

---

# 📌 À savoir

Le fournisseur IA n'influence pas le fonctionnement des autres modules d'APO Studio.

Vous pouvez changer de fournisseur à tout moment sans modifier vos projets existants.
