# ✂️ SmartCut

SmartCut est le système de découpage intelligent d'APO Studio.

Son objectif est de transformer automatiquement une longue VOD en plusieurs épisodes YouTube cohérents.

Au lieu de couper la vidéo toutes les 30 minutes, SmartCut analyse son contenu afin de trouver les meilleurs moments pour créer une coupure naturelle.

---

# 🧠 Comment fonctionne SmartCut ?

SmartCut s'appuie sur la transcription générée par Whisper.

L'intelligence artificielle analyse ensuite le contenu afin de repérer les changements importants dans la vidéo :

- changement d'objectif ;
- fin d'un événement ;
- changement de zone ou d'activité ;
- pause naturelle dans la discussion.

Ces moments sont utilisés pour proposer des coupures plus agréables à regarder.

---

# 🎬 Une série plus naturelle

L'objectif n'est pas de produire des épisodes ayant exactement la même durée.

À la place, SmartCut privilégie une bonne expérience de visionnage.

Un épisode peut donc être légèrement plus court ou plus long si cela permet d'obtenir une meilleure transition.

---

# ⚙️ Durée cible

Lors de la création du projet, vous choisissez une durée cible pour vos épisodes.

Cette durée sert uniquement de référence.

SmartCut reste libre de décaler légèrement une coupure lorsqu'il détecte un meilleur moment.

---

# 🤖 Décisions de l'IA

Les propositions de découpage sont réalisées automatiquement.

L'IA cherche à éviter les coupures :

- au milieu d'une phrase ;
- pendant un combat important ;
- en pleine explication ;
- au milieu d'un objectif.

Elle privilégie toujours les transitions naturelles.

---

# 💡 Conseils

SmartCut fonctionne particulièrement bien sur :

- les jeux narratifs ;
- les RPG ;
- les jeux de gestion ;
- les jeux d'aventure.

Pour les jeux très rapides ou les contenus très dynamiques, les propositions peuvent être légèrement différentes selon le contexte.
