# APO Studio

APO Studio est un logiciel Windows permettant d'automatiser la création de vidéos YouTube à partir de VOD Twitch grâce à l'intelligence artificielle.

## Fonctionnalités

- Création automatique de projets
- Transcription Whisper
- Génération IA
- Découpage intelligent (SmartCut)
- Génération des miniatures
- Workflow personnalisable
- Support multi IA (OpenAI, Claude, GLM, Ollama)

## Technologies

- Python
- CustomTkinter
- Whisper / Faster Whisper
- FFmpeg
